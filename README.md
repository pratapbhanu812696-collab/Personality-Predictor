# AI Personality Predictor

A simple AI personality predictor built with `customtkinter`. It asks a series of questions and simulates a personality analysis for Extraversion, Conscientiousness, and Agreeableness.

## Project Structure
- `main.py`: The entry point to run the application.
- `app.py`: Contains the `PersonalityApp` UI class.
- `logic.py`: Contains the logic for calculating personality results based on answers.
- `data.py`: Contains the questions used in the quiz.

## Requirements
- Python 3.x
- `customtkinter`

## Installation & Running
1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   python main.py
   ```
