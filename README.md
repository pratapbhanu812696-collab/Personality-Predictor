# 🧠 AI Personality Predictor

Ek simple aur interactive application jo **Big Five Personality Traits** ke aadhar par aapki personality analyze karti hai. Is project mein Desktop (GUI) aur Web (Streamlit) dono versions shamil hain.

## 🚀 Live Demo
Aap is app ko browser mein yahan chala sakte hain:
👉 **[Live Web App on Render](https://personality-predictor-0cfy.onrender.com)**

---

## ✨ Features
- **Interactive UI:** Modern dark-mode interface.
- **Real-time Analysis:** Aapke answers ke aadhar par turant results.
- **Cross-Platform:** Browser par bhi chalta hai aur desktop par bhi.
- **Big Five Model:** Extraversion, Agreeableness, Conscientiousness, Neuroticism, aur Openness ka calculation.

---

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Web Framework:** [Streamlit](https://streamlit.io/) (For Web Deployment)
- **Desktop UI:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) (For Desktop Version)
- **Deployment:** [Render](https://render.com/)

---

## 💻 Local Machine par Kaise Chalayein?

### 1. Repository ko Clone karein
```bash
git clone [https://github.com/YOUR_USERNAME/Personality-Predictor.git](https://github.com/YOUR_USERNAME/Personality-Predictor.git)
cd Personality-Predictor
2. Dependencies Install karein
Bash
pip install -r requirements.txt
3. App Run karein
Web Version ke liye:

Bash
streamlit run app.py
Desktop GUI ke liye:

Bash
python personality_gui.py
📊 Personality Traits Explained
Yeh app niche diye gaye 5 traits ko measure karti hai:

Extraversion: Aap kitne social aur energetic hain.

Agreeableness: Aap doosron ke prati kitne dayalu aur sahmat hain.

Conscientiousness: Aap kitne organized aur zimmedar hain.

Neuroticism: Aap stress ko kaise handle karte hain.

Openness: Aap nayi cheezon aur creativity ke liye kitne khule hain.
